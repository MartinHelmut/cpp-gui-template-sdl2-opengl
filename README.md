# Base GUI project setup with SDL2 and OpenGL

C++ gui project template with SDL2 and OpenGL 4.1.

## Setup

This project requires [Litr](https://github.com/krieselreihe/litr).

Quick setup to build and run the application:

```shell
litr build,start
```
